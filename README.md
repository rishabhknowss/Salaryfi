# Salaryfi

Companies pay employees in crypto . A modern payroll solution for remote work, providing secure, private salary payments.


## Installation

Clone Repository

```bash
  git clone https://github.com/rishabhknowss/Salaryfi.git
```
Installation


```bash
  cd my-project
  npm install
```

```bash
 npm start
```
